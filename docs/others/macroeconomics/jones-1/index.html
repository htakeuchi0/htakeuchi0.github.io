    ジョーンズ マクロ経済学Ⅰ #

    概要 #

    手取りを増やすことを公約に掲げた政党の台頭が示すように，昨今，日本経済再生への期待が高まっています． 本ページは，国の経済全体に関する経済学についてのノートです．

    経済を構成する個々の主体に着目するミクロ経済学に対し，それらを合わせて一国の経済全体に着目する経済学の領域は マクロ経済学 と呼ばれます． 本ページでは，作成者が『ジョーンズ マクロ経済学Ⅰ 長期成長編』[1] を初読した際のメモをまとめます． 本ページの作成者は，[1] を読むまで，経済学に関する知識を一切有していませんでした．

    本教科書 [1] は，簡単な数式を使って説明がなされており，数式に抵抗がない初心者にとってはわかりやすく感じました． 本ページでは，数式で表現された箇所を中心に，初読時のメモを整理して記載します．

    本ページは解説文ではなく，あくまでメモの整理として数式で表されたモデルを中心にまとめますが，本教科書 [1] は数式ばかりがでてくるものではありません． また，本文中にもあるように，マクロ経済学は数式のみから議論・検討を行う学問でないこともお断りしておきます．

    第1章 マクロ経済学への招待 #

    マクロ経済学の方法は以下のとおりです．

    1. 事実を記述する．
    2. モデル を開発する．
    3. モデルから導き出される予見と元の事実とを比較する．
    4. モデルを使って他のケースの予見を行う．

    本ページでは，特に項番2にあるように，どのようなモデルが知られているかに着目してまとめます．

    モデルの入力は パラメータ と 外生変数 で，出力は 内生変数 であると整理されます．

    パラメータ -----> モデル ----> 内生変数
外生変数
    • パラメータ：モデルの外から与える定数
    • 外生変数：モデルの外から与える変数（時間的変化を許す）
    • 内生変数：未知数に相当する変数

    以下，外から与えられるものを \(\bar{\cdot}\) と上線を付けた記号で表します．

    第2章 マクロ経済学の計測 #

    GDP とは国内生産とのことです． 生産＝支出＝所得 の関係が基本的原則で，それぞれの観点からGDPを計算しても同じ値となります．

    GDP の支出アプローチでは，GDP は以下の式で表されます．

    \[ Y=C+I+G+\mathit{NX} \]

    記号の意味は以下のとおりです．

    • \(Y\) : GDP
    • \(C\) : 消費
    • \(I\) : 投資
    • \(G\) : 政府支出
    • \(\mathit{NX}\) : 純輸出

    純輸出は 貿易・サービス収支 のことで， \(<0\) のとき 貿易赤字 と呼ばれるものです．

    GDP は 名目GDP という考え方と 実質GDP という考え方があり，以下の関係があります．

    \[ \text{名目GDP}=\text{価格水準}\times\text{実質GDP} \]

    名目GDPは価格レベルの表現で，実質GDPは数量レベルの表現といえます． ただし，実質GDPも特定の年の価格で表すことがふつうです． 年をまたぐときに，どの年で統一するかにはいくつかの流儀があります．

    • ラスパイレス指数：基準時点（当初）で表す．
    • パーシェ指数：比較時点（終わり）で表す．
    • 連鎖指数：各指数の倍率の相乗平均で表す．

    価格水準 は GDPデフレータ と呼ばれることもあります． また，価格水準の変化率は インフレ率 と呼ばれます．

    第3章 経済成長の概観 #

    人類がこの世に登場してからの13万年を1日に圧縮すると，近代経済成長の時代はわずか最後の3分で，急激な成長といえます． また，この近代経済成長の開始のタイミングの違いが経済発展水準の格差拡大を生んでいます．

    この 経済成長 をどう定義するかですが，これは1人あたりのGDPの変化率と捉えることができます．

    成長率については一般に次のことが成り立ちます． \(\bar{g}\) を成長率とし， \(y_t\) を時刻 \(t\) における変数 \(y\) の値とします． このとき，

    \[ \begin{aligned} &y_t=y_0(1+\bar{g})^t,\\ &\bar{g}=\left(\frac{y_t}{y_0}\right)^{1/t}-1 \end{aligned} \]

    が成り立ちます． 後者を 成長率の計算公式 と呼びます．

    いま， \(y_t=2y_0\) となる \(t\) を求めることを考えます． \((1+\bar{g})^t=2\) なので， \(t=\ln 2/\ln(1+\bar{g})\) ですが， \(\bar{g}\ll1\) ならば \(\ln(1+\bar{g})\approx\bar{g}\) なので， \(t\approx\ln 2/\bar{g}\approx0.7/\bar{g}\) となります．

    つまり，1年に2%ずつ成長するなら \(70\%/2\%=35\) なので，およそ35年で2倍になることがわかります．

    この法則を 成長に関する70の法則 といいます．

    成長率の公式については次のことも言えます．

    \(x,y,z\) を変数とし，その成長率を \(g_x,g_y,g_z\) とします． つまり， \(x=(x_t),y=(y_t),z=(z_t)\) とし，

    \[ \begin{aligned} &x_{t+1}=(1+g_x)x_t,\\ &y_{t+1}=(1+g_y)y_t,\\ &z_{t+1}=(1+g_z)z_t \end{aligned} \]

    とします．

    いま， \(xy=(x_ty_t)\) で定義し， \(x=y\) を任意の \(t\) について \(x_t=y_t\) であると定義します． ほかの演算（ \(x/y\) など）も同様とします．

    \(x_0,y_0,z_0>0\) としたとき， \(z=xy\) ならば，

    \[ \frac{z_{t+1}}{z_t}=\frac{x_{t+1}}{x_t}\frac{y_{t+1}}{y_t} \]

    なので，

    \[ 1+g_z=(1+g_x)(1+g_y)=1+(g_x+g_y)+g_xg_y \]

    となります． ここで， \(g_x,g_y\ll1\) なら \(g_xg_y\approx0\) なので，

    \[ g_z=(g_x+g_y)+g_xg_y\approx g_x+g_y \]

    が成り立ちます． つまり， \(g_x,g_y\ll1\) ならば，

    \[ z=xy\implies g_z\approx g_x+g_y \]

    同様の仮定のもとで，

    \[ \begin{aligned} &z=x/y\implies g_z\approx g_x-g_y,\\ &z=x^a\implies g_z\approx ag_x \end{aligned} \]

    がいえます．

    第4章 生産のモデル #

    以下のモデルを コブ=ダグラス型生産関数 と呼びます．

    \[ Y=F(K,L)=\bar{A}K^{1/3}L^{2/3} \]

    ただし，

    • \(Y\) : 生産量
    • \(K\) : 資本
    • \(L\) : 労働力
    • \(\bar{A}\) : 生産性パラメータ

    です．

    資本の限界生産物 \(\mathit{MPK}\) は \[ \mathit{MPK}=\frac{\partial Y}{\partial K}=\frac{1}{3}\bar{A}K^{-2/3}L^{2/3}=\frac{1}{3}\frac{Y}{K} \] と表されます．

    労働の限界生産物 \(\mathit{MPL}\) は \[ \mathit{MPL}=\frac{\partial Y}{\partial L}=\frac{2}{3}\bar{A}K^{1/3}L^{-1/3}=\frac{2}{3}\frac{Y}{L} \] と表されます．

    賃金率 \(w\) , 資本のレンタル価格 \(r\) に対し，以下の最適化問題を考えます．

    \[ \text{maximize\ }\Pi=F(K,L)-rK-wL. \]

    ここで，

    • \(\mathit{MPK}\ge r\) なら資本投入
    • \(\mathit{MPL}\ge w\) なら労働投入

    なる戦略をとるとき， \(\bar{K},\bar{L}\) を所与とすると，以下の生産のモデルが立てられます．

    \[ \begin{cases} Y=\bar{A}K^{1/3}L^{2/3},\\ \dfrac{1}{3}\dfrac{Y}{K}=r,\\[8pt] \dfrac{2}{3}\dfrac{Y}{L}=w,\\ K=\bar{K},\\ L=\bar{L}. \end{cases} \]

    この方程式系の解は以下のとおりで，解のことを 均衡 と呼びます．

    \[ \begin{cases} K^*=\bar{K},\\ L^*=\bar{L},\\ r^*=\dfrac{1}{3}\bar{A}\left(\dfrac{\bar{L}}{\bar{K}}\right)^{2/3},\\[8pt] w^*=\dfrac{2}{3}\bar{A}\left(\dfrac{\bar{K}}{\bar{L}}\right)^{1/3},\\ Y^*=\bar{A}\bar{K}^{1/3}{L}^{2/3}. \end{cases} \]

    \(y=Y/L\) , \(k=K/L\) とおくと，

    • \(y\) : 1人あたりの産出量
    • \(k\) : 1人あたりの資本

    となりますが，

    \[ y^*=\frac{Y^*}{L^*}=\frac{\bar{A}\bar{K}^{1/3}\bar{L}^{2/3}}{\bar{L}}=\bar{A}\left(\frac{\bar{K}}{\bar{L}}\right)^{1/3}=\bar{A}\bar{k}^{1/3} \]

    が成り立ちます． \(\bar{A}\) は 全要素生産性 (TFP) といい，逆に以下から求めることもできます． \[ \bar{A}=\frac{y^*}{\bar{k}^{1/3}}. \]

    A国，B国の比較では以下のように計算することができます． \[ \frac{y^*_{\mathrm{A}}}{y^*_{\mathrm{B}}}=\frac{\bar{A}_{\mathrm{A}}}{\bar{A}_{\mathrm{B}}}\left(\frac{\bar{k}_{\mathrm{A}}}{\bar{k}_{\mathrm{B}}}\right)^{1/3}. \]

    第5章 ソロー・モデル #

    コブ=ダグラス型生産関数から ソロー・モデル を構築します．

    コブ=ダグラス型関数とは次のようなものでした．

    \[ Y=F(K,L)=\bar{A}K^{1/3}Y^{2/3} \]

    ただし，

    • \(Y\) : 生産量
    • \(K\) : 資本
    • \(L\) : 労働力
    • \(\bar{A}\) : 生産性パラメータ

    です．

    これを時刻 \(t\) ごとの方程式に置き換えると以下のようになります．

    \[ Y_t=F(K_t,L_t)=\bar{A}K_t^{1/3}L_t^{2/3} \]

    次に，生産物と消費，投資との関係を以下のように表します．

    \[ C_t+I_t=Y_t \]

    ただし，

    • \(C_t\) : 消費
    • \(I_t\) : 投資

    です． \(Y_t-C_t\) は 貯蓄 に対応し， \(Y_t-C_t=I_t\) なので，貯蓄額が投資額に一致することを表しています．

    次に 資本蓄積方程式 を以下のように与えます．

    \[ \Delta K_{t+1}=I_t-\bar{d}K_t. \]

    ただし，

    • \(\bar{d}\) : 資本減耗率

    です． また， \(\Delta K_{t+1}=K_{t+1}-K_t \) として，他の変数でも同様の意味とします． ここで， \(K_t\) は 資本ストック， \(\Delta K_{t+1}\) は 資本ストックの変化量 と捉えられます． この式は，

    \[ K_{t+1}=K_t+I_t-\bar{d}K_t. \]

    と捉えたほうがわかりやすいかもしれません． \(\bar{d}K_t\) は 資本減耗 の量を表します．

    なお，ストック とは期を超えて存在し続けるものを指しますが，これに対してひと期間しか持続しない量は フロー と呼ばれます．

    次に，投資への割合を \[ I_t=\bar{s}Y_t \] とします．ただし，

    • \(\bar{s}\) : 投資率

    とします．

    労働供給が一定とすると \(L_t=\bar{L}\) とできるので，以上をまとめると次の方程式系が得られます． \[ \begin{cases} Y_t=F(K_t,L_t)=\bar{A}K_t^{1/3}L_t^{2/3},\\ \Delta K_{t+1}=I_t-\bar{d}K_t,\\ L_t=\bar{L},\\ C_t+I_t=Y_t,\\ I_t=\bar{s}Y_t. \end{cases} \] これを ソロー・モデル といいます．

    \(\Delta K_{t+1}=0\) のときを 定常状態 といい， \(K_t=K_{t+1}=\cdots\) となる \(t\) を用いて \(K^*=K_t,Y^*=Y_t\) などとします．

    \(0=\Delta K_{t+1}=I_t-\bar{d}K_t=\bar{s}Y_t-\bar{d}K_t=\bar{s}Y^*-\bar{d}K^*\) なので \(\bar{s}Y^*=\bar{d}K^*\) です． これは， \[ \frac{K^*}{Y^*}=\frac{\bar{s}}{\bar{d}} \] であることも表しています．

    また， \(Y^*=\bar{A}{K^*}^{1/3}\bar{L}^{2/3}\) なので， \[ \frac{\bar{d}}{\bar{s}}K^*=\bar{A}{K^*}^{1/3}\bar{L}^{2/3}\\ \] から \[ {K^*}^{2/3}=\frac{\bar{s}\bar{A}}{\bar{d}}\bar{L}^{2/3}\\ \] なので \[ K^*=\left(\frac{\bar{s}\bar{A}}{\bar{d}}\right)^{3/2}\bar{L} \] であることがわかりました．

    同様に， \[ \begin{aligned} Y^*&=\bar{A}{K^*}^{1/3}\bar{L}^{2/3}\\ &=\bar{A}\left(\frac{\bar{s}\bar{A}}{\bar{d}}\right)^{1/2}\bar{L}^{1/3}\bar{L}^{2/3}\\ &=\left(\frac{\bar{s}}{\bar{d}}\right)^{1/2}\bar{A}^{3/2}\bar{L} \end{aligned} \] であることがわかります．

    また， \(L^*=\bar{L}\) なので， \[ y^*=\frac{Y^*}{L^*}=\bar{A}^{3/2}\left(\frac{\bar{s}}{\bar{d}}\right)^{1/2} \] です．

    ソロー・モデルからは 移行過程の動学原理 と呼ばれる原理が導かれます． これは，定常状態より低い所得水準にあればあるほど成長率は大きいという原理です． この原理は資本蓄積方程式から導かれます．

    資本蓄積方程式から \(\Delta K_{t+1}=I_t-\bar{d}K_t\) ですが， \(I_t=\bar{s}Y_t\) なので，

    \[ \frac{\Delta K_{t+1}}{K_t}=\bar{s}\frac{Y_t}{K_t}-\bar{d} \]

    となります（ただし \(K_t>0\) とします）．

    さらに， \(\bar{s}Y^*=\bar{d}K^*\) なので，

    \[ \begin{aligned} &\frac{\Delta K_{t+1}}{K_t}=\bar{s}\frac{Y_t}{K_t}-\bar{d}\\ &=\bar{S}\left(\frac{Y_t}{K_t}-\frac{Y^*}{K^*}\right)\\ &=\bar{s}\frac{Y^*}{K^*}\left(\frac{Y_t/K_t}{Y^*/K^*}-1\right)\\ &=\bar{s}\frac{Y^*}{K^*}\left(\frac{\bar{A}K_t^{1/3}\bar{L}^{2/3}/K_t}{\bar{A}{K^*}^{1/3}\bar{L}^{2/3}/K^*}-1\right)\\ &=\bar{s}\frac{Y^*}{K^*}\left(\frac{K_t^{-2/3}}{{K^*}^{-2/3}}-1\right)\\ &=\bar{s}\frac{Y^*}{K^*}\left(\frac{{K^*}^{2/3}}{K_t^{2/3}}-1\right) \end{aligned} \]

    となります．

    よって \(K_t < K^*\) ならば資本ストック \(K_{t+1}\) は増加し，その成長率は \(K_t\ll K^*\) の方が大きくなります． さらに，生産過程を通じて，資本ストック \(K_t\) は \(K^*\) に向かい，その後はそこに留まり続けることを示しています．

    第6章 アイディアと経済成長 #

    ソロー・モデルには，長期的な成長を説明する理論的な枠組みが欠けている欠点があります． そこで，モノに加えて アイディア も考えることで，そのような理論が構築できることが知られています． アイディアは競合性がないことが重要です． そのモデルは ローマー・モデル と呼ばれています．

    ローマー・モデルを次のように構築します． まず，以下の方程式を考えます．

    \[ Y_t=A_tL_{yt}. \]

    ただし，

    • \(Y_t\) : 生産量
    • \(A_t\) : 知識ストック
    • \(L_t\) : 労働投入量

    を表します．

    労働ストックの変化を以下のように表します．

    \[ \Delta A_{t+1}=\bar{z}A_tL_{at}. \]

    ただし，

    • \(\bar{z}\) : 生産性パラメータ
    • \(L_{at}\) : アイディア開発のために投入される労働量

    を表します．また，

    \[ \begin{aligned} &L_{yt}+L_{at}=\bar{L},\\ &L_{at}=\bar{\ell}\bar{L} \end{aligned} \]

    とします． ただし，

    • \(\bar{\ell}\) : アイディア開発に従事する労働量の割合

    を表します．

    以上をまとめると以下のようになります． \[ \begin{cases} Y_t=A_tL_{yt},\\ \Delta A_{t+1}=\bar{z}A_tL_{at},\\ L_{yt}+L_{at}=\bar{L},\\ L_{at}=\bar{\ell}\bar{L} \end{cases} \]

    これを ローマー・モデル といいます．

    ローマー・モデルの解を求めます．

    \(y_t=Y_t/\bar{L}\) とすると，

    \[ \begin{aligned} y_t&=\frac{Y_t}{\bar{L}}\\ &=\frac{A_tL_{yt}}{\bar{L}}\\ &=\frac{A_t(1-\bar{\ell})\bar{L}}{\bar{L}}\\ &=A_t(1-\bar{\ell}) \end{aligned} \]

    となるので，1人あたりの生産量は \(A_t\) に比例することがわかります． また，

    \[ \frac{\Delta A_{t+1}}{A_t}=\bar{z}L_{at}=\bar{z}\bar{\ell}\bar{L} \]

    なので， \(\bar{g}=\bar{z}\bar{\ell}\bar{L}\) とおくと

    \[ A_t=\bar{A}_0(1+\bar{g})^t \]

    となり，

    \[ y_t=\bar{A}_0(1-\bar{\ell})(1+\bar{g})^t \]

    と表せます．

    さて，ソロー・モデルは以下のようなもので，アイディアが考慮されていないモデルでした．

    \[ \begin{cases} Y_t=F(K_t,L_t)=\bar{A}K_t^{1/3}L_t^{2/3},\\ \Delta K_{t+1}=I_t-\bar{d}K_t,\\ L_t=\bar{L},\\ C_t+I_t=Y_t,\\ I_t=\bar{s}Y_t. \end{cases} \]

    ローマー・モデルは以下のようなもので，アイディアは考慮しますが資産については省略したモデルでした．

    \[ \begin{cases} Y_t=A_tL_{yt},\\ \Delta A_{t+1}=\bar{z}A_tL_{at},\\ L_{yt}+L_{at}=\bar{L},\\ L_{at}=\bar{\ell}\bar{L} \end{cases} \]

    そこで，これらを統合し，資産とアイディアを両方考慮するモデルが考えられます． このモデルを ソロー=ローマー統合モデル と呼びます．

    ソロー=ローマー統合モデルは以下のように表されます．

    \[ \begin{cases} Y_t=A_tK_t^{1/3}L_{yt}^{2/3},\\ \Delta K_{t+1}=\bar{s}Y_t-\bar{d}K_t,\\ \Delta A_{t+1}=\bar{z}A_tL_{at},\\ L_{yt}+L_{at}=\bar{L},\\ L_{at}=\bar{\ell}\bar{L}. \end{cases} \]

    ソロー=ローマー統合モデルの解を求めます．

    変数 \(X=(X_t)\) について， \(t\) における成長率 \(g_{X_t}\) を

    \[ g_{X_t}=\frac{\Delta X_{t+1}}{X_t} \]

    で定めます． すると， \[ \begin{aligned} &g_{Y_t}=g_{A_t}+\frac{1}{3}g_{K_t}+\frac{2}{3}g_{L_{yt}},\\ &g_{A_t}=\frac{\Delta A_{t+1}}{A_t}=\bar{z}\bar{\ell}\bar{L}=\bar{g},\\ &g_{K_t}=\frac{\Delta K_{t+1}}{K_t}=\bar{s}\frac{Y_t}{K_t}-\bar{d} \end{aligned} \] が得られます．

    よって，資本ストック \(K_t\) の成長率 \(g_{K_t}\) が一定になる場合を考えると， \(Y_t/K_t\) が一定となる必要があることがわかります． つまり， \(g_K^*=g_Y^*\) です． また，労働量が一定である仮定をおくと \(g_{L_{yt}}=0\) となります． このとき， \[ g_Y^*=\bar{g}+\frac{1}{3}g_K^*+\frac{2}{3}\cdot0=\bar{g}+\frac{1}{3}g_Y^* \] なので， \[ g_Y^*=\frac{3}{2}\bar{g} \] となります．

    さて， \(g_K^*=g_Y^*\) なので， \[ \frac{K_t^*}{Y_t^*}=\frac{\bar{s}}{g_K^*+\bar{d}}=\frac{\bar{s}}{g_Y^*+\bar{d}} \] です． そして， \[ Y_t^*=A_t^*\left(\frac{\bar{s}Y_t^*}{g_Y^*+\bar{d}}\right)^{1/3}L_{yt}^{2/3} \] なので， \[ {Y_t^*}^{2/3}=A_t^*\left(\frac{\bar{s}}{g_Y^*+\bar{d}}\right)^{1/3}L_{yt}^{2/3} \] より， \[ \begin{aligned} Y_t^*&={A_t^*}^{3/2}\left(\frac{\bar{s}}{g_Y^*+\bar{d}}\right)^{1/2}L_{yt}\\ &={A_t^*}^{3/2}\left(\frac{\bar{s}}{g_Y^*+\bar{d}}\right)^{1/2}\bar{L}(1-\bar{\ell}) \end{aligned} \] です． よって， \[ \begin{aligned} y_t^*&=\frac{Y_t^*}{\bar{L}}\\ &={A_t^*}^{3/2}\left(\frac{\bar{s}}{g_Y^*+\bar{d}}\right)^{1/2}(1-\bar{\ell}) \end{aligned} \] と求まります．

    第7章 労働市場，賃金，および失業 #

    失業は以下のように分けられます．

    \[ \text{失業}=\text{自然的}+\text{循環的}=(\text{摩擦的}+\text{構造的})+\text{循環的} \]
    • 摩擦的：経済環境の変化により労働者が食を変更するための失業
    • 構造的：労働者と企業のマッチングによる失業
    • 循環的：短期変動に関連した失業

    以下を バスタブ・モデル と呼びます．

    \[ \begin{cases} E_t+U_t=\bar{L},\\ \Delta U_{t+1}=\bar{s}E_t+\bar{f}U_t \end{cases} \]

    ただし，

    • \(E_t\) : 就業者
    • \(U_t\) : 失業者
    • \(\bar{s}\) : 離職率
    • \(\bar{f}\) : 新規就職率

    という意味です．

    定常状態は \(\Delta U_{t+1}=0\) の状態です． このとき， \[ \begin{aligned} 0&=\bar{s}E_t-\bar{f}U_t\\ &=\bar{s}(\bar{L}-U_t)-\bar{f}U_t\\ &=\bar{s}\bar{L}-(\bar{f}+\bar{s})U_t \end{aligned} \] なので， \[ U^*=\frac{\bar{s}\bar{L}}{\bar{f}+\bar{s}} \] となります． または， \[ u^*=\frac{U^*}{\bar{L}}=\frac{\bar{s}}{\bar{f}+\bar{s}} \] といえます．

    このバスタブ・モデルという呼称は，蛇口と排水溝をもつ入れ物と水かさの関係で例えたもので，バスタブ曲線のようにグラフの形状を表すものではありません．

    第8章 インフレーション #

    貨幣数量説 は以下で表されます．

    \[ M_tV_t=P_tY_t \]

    ただし，

    • \(M_t\) : 貨幣量
    • \(V_t\) : 貨幣の流通速度
    • \(P_t\) : 物価水準
    • \(Y_t\) : 実質GDP

    を表します．

    ここで， \(Y_t=\bar{Y}_t,V_t=\bar{V},M_t=\bar{M}_t\) は所与としてよいと考えられるので，

    \[ P_t^*=\frac{\bar{M}_t\bar{V}}{\bar{Y}_t} \]

    と表せます．

    \(\bar{g}_M+\bar{g}_V=\bar{g}_P+\bar{g}_Y\) としたとき，流通速度一定とすると \(\bar{g}_V=0\) となります．

    インフレ率を \(\pi\) で表すことが多いので， \(\bar{g}_P\) （物価水準の変化率＝インフレ率）を \(\pi\) で表すと， \[ \pi=\bar{g}_M-\bar{g}_Y \] と書けます．

    また， \(\text{名目GDP}=\text{価格水準}\times\text{実質GDP}\) より， \[ \text{名目利子率}=\text{実質利子率}+\text{インフレ率} \] となり，次の記号で表されます． \[ i=R+\pi. \]
